Speichenlängen ausrechen

Messwerte manuell eingeben

Füllen sie alle Felder in allen Bereichen des Formulars aus. Das Ergebnis wird dann automatisch angezeigt. Wenn Sie irgendeinen Wert ändern, wird das Ergebnis automatisch angepasst.

Messwerte aus Datenbank lesen

Klicken Sie auf das erste Feld im Bereich Felgen und wählen Sie die gewünschte Felge aus der Liste aus. Dadurch werden die darunter liegenden Felder automatisch gefüllt. Wählen sie dann genauso im Feld Naben eine Nabe und im Feld Rechner die Anzahl der Speichenkreuzungen. Das Ergebnis wird automatisch angezeigt. Wenn Sie irgendeinen Wert ändern, wird das Ergebnis automatisch angepasst.

Datenbank bearbeiten

Neues Teil mit manuell eingegebenen Messwerten eintragen

Füllen sie alle Felder in dem Bereich des Formulars aus, bei dem Sie das Teil eintragen möchten. Sobald Sie dort alle Felder ausgefüllt haben, wird oben Rechts im Feld ein gelber Kreis mit einem Speicher-Symbol auftauchen. Klicken Sie darauf, und es wird sich ein kleines Pop-Up-Fenster öffnen. Tragen Sie in diesem Feld den Namen und die Marke des entsprechenden Teils ein. Der Name darf nicht schon in der Datenbank vorhanden sein, und kann auch eine Beschreibung des Teils enthalten. Zum Beispiel:

Marke Name
Ambrosio V-Profil ohne Ösen 622-13,5
H+Son EERO 559
Ryde Andra 210, ungeöst, gepunzt

Klicken sie danach darunter auf "Speichern". Das Neue Teil ist jetzt in der Datenbank verfügbar.

Neues Teil mit vorhandenem Teil als Vorlage eintragen

Wählen Sie das Teil, das als Vorlage dienen soll so an, als wollten Sie es zum Ausrechen von Speichenlängen benutzen (s.o.). Es wird, wie wenn Sie die Werte manuell eintragen, ein gelber Kreis mit einem Speichern-Symbol auftauchen. Klicken sie diesen an und klicken sie dann auf "Als neue Felge/Nabe Speichern...", und geben sie wie oben beschrieben die Marke und den Namen ein und klicken sie auf "Speichern".

Werte von Vorhandenem Teil ändern

Wählen Sie das Teil, das Sie ändern wollen so an, als wollten Sie es zum Ausrechen von Speichenlängen benutzen (s.o.). Nehmen Sie nun in den entsprechenden Feldern die gewünschten änderungen vor. Klicken sie danach auf den gelben Kreis mit dem Speichern-Symbol oben rechts im Feld, und danach auf "Überschreiben". Falls die Änderungen nicht sofort eintreten, warten Sie einige Sekunden und starten Sie SpeiLang 2 neu.

Teil aus der Datenbank löschen

Wählen Sie das Teil, das Sie löschen wollen so an, als wollten Sie es zum Ausrechen von Speichenlängen benutzen (s.o.). Klicken sie dann auf den roten Kreis mit dem Mülltonnen-Symbol oben rechts im Feld, und danach auf "Löschen".

Teile, Werte & Namen direkt in der Datenbank ändern

Um die Datenbankoberfläche zu öffnen, klicken Sie unten rechts auf den Blauen Kreis mit dem Zahnrad darauf. Es öffnet sich das Einstellungen-Fenster. Klicken Sie hier auf "Datenbankzugriff", und es wird sich ein Browserfenster öffnen.

Sie sind nun im CouchDB - Fauxton. Im Hauptfenster kommen Sie direkt zu den einzelnen Datenbanken.

Auf den Datenbanken-Seiten sind alle Einträge auf Seiten aufgeteilt aufgelistet. Wenn Sie mit der Maus über einen Eintrag fahren, erscheinen oben rechts im Eintra die Optionen, den Eintrag (Das "Dokument") zu löschen oder es als JSON zu beabeiten.

Erweitert

Einstellungen der Datenbank ändern

Um die Datenbankoberfläche zu öffnen, klicken sie unten rechts auf den Blauen Kreis mit dem Zahnrad darauf. Es öffnet sich das Einstellungen-Fenster. Klicken Sie hier auf "Datenbankzugriff", und es wird sich ein Browserfenster öffnen.

Auf der Linken Seite kommen Sie zu den Einstellungen der Datenbanken. Dort können Sie oben rechts Einträge hinzufügen oder in jeder Zeile den entsprechenden Eintrag bearbeiten oder entfernen.

Datenbank in Datei Exportieren

Klicken Sie unten rechts auf den Blauen Kreis mit dem Zahnrad darauf. Es öffnet sich das Einstellungen-Fenster. Wenn Sie jetzt auf "Exportieren" klicken, wird die Datenbank als JSON-Datei zum Download angeboten.

Datenbank aus Datei Importieren

ACHTUNG! Dies wird die momentan geladene Datenbank komplett vernichten und durch die aus der Datei ersetzen!

Klicken Sie unten rechts auf den Blauen Kreis mit dem Zahnrad darauf. Es öffnet sich das Einstellungen-Fenster. Klicken Sie jetzt auf "Importieren" und es öffnet sich Ein Fenster, in dem Sie die hochzuladende Datei auswählen können. Wenn Sie sie ausgewählt haben, klicken Sie auf "OK". Bestätigen Sie die Sicherheitsnachfrage mit "OK". Sobald sich das Einstellungen-Fenster schließt sollten die Listen aktualisiert sein. Wenn nicht, warten sie ein paar Sekunden und öffnen Sie SpeiLang 2 neu.

SpeiLang 2 Einrichten

Um SpeiLang einzurichten müssen folgende Dinge getan werden:

  1. CouchDB herunterladen und installieren von http://couchdb.apache.org/#download
  2. CORS aktivieren (siehe http://pouchdb.com/getting-started.html#enabling_cors
  3. IP-listening erweitern: Datei C:\Program Files (x86)\Apache Software Foundation\CouchDB\etc\couchdb\local.ini öffnen und in Sektion [httpd] "bind_address" einkommentieren und auf wert "0.0.0.0" setzen, sodass die Zeile "bind_address = 0.0.0.0" lautet.
  4. Erlaubende eingehende Regel für Port 5984 in Firewall einstellen
  5. SpeiLang 2 Herunterladen von https://github.com/GladOskar/SpeiLang-2/ (Unten Rechts --> "Download ZIP")
  6. Entpacken in Verzeichnis der Wahl
  7. Desktopverknüpfung erstellen (Ziel: index.html, Icon: faviconW.ico)
  8. In SpeiLang 2 über "Einstellungen" > "Importieren" die Datei \devres\db_files\CorrectForImport.json Importieren
  9. Fertig!

Falls die Datenbank auf einem anderen Computer liegt, müssen nur die Schritte 4 bis 7 durchgeführt werden, und danach über "Einstellungen" > "Server-IP" die IP des Servers eingegeben und auf "Setzen" geklickt werden.
